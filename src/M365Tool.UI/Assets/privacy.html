概述

感谢您选择使用 21Vianet Office工具箱（以下简称“本工具”）。本工具是一款面向 21Vianet / 21V Microsoft 365 运维场景的 Windows 桌面工具，用于辅助完成 Microsoft 365 Apps 安装、Office 卸载、更新频道切换、激活痕迹清理、Teams 常见问题处理、Outlook 诊断以及相关本地维护操作。

我们深知账户信息、登录状态和本地诊断数据对您至关重要。本隐私政策旨在说明本工具在使用过程中会处理哪些信息、出于何种目的处理这些信息、信息保存在哪里，以及我们不会收集或保存哪些敏感信息。

请您在使用本工具前仔细阅读本隐私政策。如您继续使用本工具，即表示您已了解并接受本政策中说明的数据处理方式。

一般原则

什么是个人信息？

个人信息是指以电子或者其他方式记录的，与已识别或者可识别的自然人有关的各种信息。本工具可能涉及的个人信息包括账号标识、显示名称、头像缓存信息、验证时间、本地配置状态以及您主动导出的诊断日志中可能包含的设备或软件运行信息。

什么是处理？

“处理”是指对个人信息进行的收集、存储、使用、加工、传输、提供、删除等操作。本工具对相关信息的处理主要发生在用户本机，用于完成 21V 账户验证状态展示、本地功能访问判断、界面语言偏好保存以及工具运行诊断。

谁负责本工具？

本工具由项目维护者维护并发布。

我们处理哪些信息

本工具遵循“本地处理、最小必要、用户可控”的原则，仅在实现功能所必需的范围内处理信息。

账户验证状态信息

用于展示当前验证状态、账号显示信息和本地功能访问判断。

本地工具配置

用于保存界面语言、页面状态、功能状态等用户偏好。

头像缓存信息

用于在工具界面中展示用户头像或账号信息。

诊断与日志信息

仅在用户主动执行相关功能时用于本地排查和结果确认。

账户验证状态信息

为确认用户是否已完成 21V 账户验证，并用于在界面中展示当前登录状态，本工具可能会在本机用户配置目录中保存以下最小化验证状态信息：

  • 界面语言设置；
  • 是否已完成 21V 验证；
  • 上次验证账号；
  • 用户显示名称；
  • 头像缓存信息；
  • 验证时间。

上述信息仅用于本工具本地界面展示和本地功能访问判断。本工具不会在本地配置文件中保存用户密码、access token、refresh token 或 Microsoft 登录 cookie。

21V 账户验证与 WebView2 登录说明

在登录过程中，用户输入的账号、密码、多因素认证信息或其他登录凭据均由 Microsoft 21V 官方登录页面处理。本工具不会读取、拦截、记录或保存用户密码。

WebView2 会话数据

登录过程中，WebView2 运行时会在当前 Windows 用户的临时目录中创建独立的浏览器用户数据目录，用于承载 Microsoft 登录页面所需的 cookie、缓存和站点存储数据。

为降低会话残留风险，本工具会在每次登录前清理 WebView2 会话数据，并在登录窗口关闭后尝试删除该临时目录。

由于 Windows 文件占用、WebView2 运行时状态或系统权限等原因，临时目录删除可能存在延迟或失败。如发生此类情况，相关数据仍位于当前 Windows 用户本机环境中，不会由本工具主动上传。

登录完成后的本地状态保存

登录完成后，本工具仅保存最小化的验证状态信息，用于界面展示和本地功能访问判断。保存内容包括界面语言、是否已完成 21V 验证、上次验证账号、显示名称、头像缓存信息以及验证时间。

该配置文件不包含用户密码、access token、refresh token 或 Microsoft 登录 cookie。

本地配置与缓存

本工具可能会在当前 Windows 用户的本地配置目录或临时目录中创建必要文件，用于保存工具配置、头像缓存、验证状态或临时运行数据。

这些信息的用途包括：

  • 显示当前验证账号和验证状态；
  • 展示用户头像或显示名称；
  • 判断是否允许访问受限功能；
  • 保存界面语言等用户偏好；
  • 支持本地诊断、安装、卸载、清理或修复流程。

本工具不会将上述本地配置和缓存信息主动上传到远程服务器。

日志与诊断信息

本工具的部分功能可能会执行 Office、Teams、Outlook 或系统环境相关的本地检查、清理、修复或诊断操作。在用户主动执行相关功能时，本工具可能会在本机生成运行日志或诊断结果，用于帮助用户判断执行结果或进行问题排查。

除非用户主动导出、复制、上传或发送相关日志，本工具不会自动将日志或诊断结果上传到远程服务器。

请注意，诊断日志可能包含设备环境、软件版本、路径、错误信息、执行时间等内容。在向他人提供日志前，建议您自行确认日志内容，避免包含不希望共享的信息。

数据保存位置与期限

本工具处理的信息主要保存在当前 Windows 用户的本机环境中，包括本地用户配置目录、临时目录或用户主动选择的日志导出位置。

本地配置文件

本地配置文件会在用户继续使用本工具期间保留，用于保持工具状态和用户体验。用户可以通过清理工具配置、删除应用数据目录或卸载工具的方式移除相关本地数据。

WebView2 临时目录

WebView2 登录会话使用的临时目录会在登录窗口关闭后由本工具尝试删除。若因系统占用或权限原因未能立即删除，用户可以在关闭本工具后手动清理对应临时目录。

日志文件

日志文件通常仅在用户主动执行诊断、导出或保存操作时产生。日志保存位置由工具默认路径或用户选择的位置决定。用户可以自行删除这些日志文件。

数据共享

本工具不会主动向任何第三方服务器上传用户密码、登录凭据、本地配置文件、验证状态文件、诊断日志或其他个人信息。

以下情况除外：

  • 用户主动将日志、截图、配置文件或诊断结果发送给支持人员、管理员或第三方；
  • 用户主动通过邮件、即时通讯工具或其他渠道提交问题信息；
  • Microsoft 21V 官方登录页面在认证过程中根据其自身机制处理登录请求、认证状态、cookie 或其他登录相关信息。

对于 Microsoft 21V 官方登录页面处理的数据，本工具无法控制其处理方式。相关数据处理应以 Microsoft 及 21Vianet 对应服务的隐私政策、服务条款和组织管理策略为准。

数据安全

本工具采取最小化保存和本地处理的方式，减少不必要的数据收集和传输。对于登录验证相关流程，本工具通过 Microsoft 21V 官方登录页面完成认证，不自行处理用户密码。

本工具会在登录前清理 WebView2 会话数据，并在登录窗口关闭后尝试删除登录过程中创建的临时浏览器用户数据目录，以降低 cookie、缓存和站点存储残留风险。

但请注意，任何本地软件运行环境都可能受到操作系统权限、恶意软件、终端安全策略、文件占用、用户误操作或第三方工具影响。建议用户仅从可信来源下载本工具，并在受控测试环境中验证后再用于生产设备。

您的权利

您可以通过以下方式管理本工具在本机保存的信息：

  • 删除本工具本地配置目录中的配置文件；
  • 删除头像缓存或临时文件；
  • 删除用户主动导出的日志文件；
  • 重新执行登录验证，以更新本地验证状态；
  • 停止使用或卸载本工具。

如果您认为本工具保存的信息不准确，或者希望删除本地保存的验证状态和配置数据，可以通过工具内提供的退出登录、清理配置功能，或手动删除对应本地文件实现。

我们不做的事

本工具明确不会执行以下行为：

  • 不收集、保存或上传用户密码；
  • 不在本地配置文件中保存 access token；
  • 不在本地配置文件中保存 refresh token；
  • 不在本地配置文件中保存 Microsoft 登录 cookie；
  • 不主动上传用户本地配置文件；
  • 不主动上传用户诊断日志；
  • 不出售用户个人信息；
  • 不使用用户信息进行广告投放；
  • 不基于用户信息进行自动化决策或画像分析；
  • 不主动向无关第三方提供用户信息。
